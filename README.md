# 2026 US Lottery Tax Rates Dataset 📊

This repository contains the most up-to-date (2026) state and federal tax rate data for US lotteries, including Powerball and Mega Millions. 

As an independent developer, I found it frustrating to hunt down the exact withholding rates, tax bracket gaps, and specific state taxes for all 50 states. So, I compiled this JSON dataset for anyone to use in their own financial apps, data analysis, or news reporting.

### 🗂 What's included in the data:
* **Federal Withholding Rates:** Automatic 24% withholding vs. the final 37% top tax bracket gap.
* **State-by-State Taxes:** 2026 state income tax rates applied specifically to lottery winnings.
* **City Taxes:** Includes special local taxes (e.g., New York City's 3.876% local tax).
* **Non-participating States:** Clarification for residents of NV, UT, HI, AL, and AK.
